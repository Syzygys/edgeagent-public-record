# EdgeAgent Public Record

EdgeAgent 的可验证 Web3 公开研究记录与 GitHub Pages 沉淀站。

- 所有含数字的发布行必须带 `source: <repo>:<path>:<line>`。
- `Web3` 仓库别名映射到本仓库根目录，详见 `SOURCE_MAP.json`。
- 内容是研究实验，不构成投资建议；在预测账本满足样本与 Brier 门槛前，不宣称存在 edge。
- 只有通过规则化审查器的既有模板内容才进入本站；新栏目或新平台首发需经过否决窗。

站点：启用 GitHub Pages 后由 `main/docs` 提供。

## 目录

- `docs/`：GitHub Pages 静态站点。
- `content/daily/`：站点文章引用的内容证据。
- `content/outbox/*/manifest.json`：机器审查结果与发布清单。
- `SOURCE_MAP.json`：来源别名解析规则。
- `project/PROJECT.md`：当前状态、验证方式与接手说明。
